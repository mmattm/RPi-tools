# RPi-tools

<aside>
➡️ Ne fonctionne que avec **ZSH**
</aside>

### Installer **SSHPass**

```bash
$ brew install sshpass
```

### Rendre les scripts executables

```bash
$ chmod +x run_syncplay.sh
$ chmod +x upload_videos.sh
```

### Executer les scripts

```bash
$ /run_syncplay.sh
$ /resume_videos.sh
```

```bash
$ /upload_videos.sh [videos_folder]
```
